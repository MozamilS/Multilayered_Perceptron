# Multilayered_Perceptron

This is a neural network project where a fully functioning neural network (multilayered perceptron) was developed from scratch following conventional neural network theory.
The neural network was built in an attempt to predict the stock market; while the model did not perform well in predicting stock market fluctuations (to the author's dismay), it was tested on other simpler problems (e.g., celsius to fahrenheit, house prices to number of rooms) and found to be reasonably trainable as a proof it works.
